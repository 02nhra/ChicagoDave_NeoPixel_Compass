ChicagoDave_NeoPixel_Compass

Arduino code, example, for compass made with Adafruit* NeoPixel Ring.

A video of working NeoPixel Compass can be seen on YouTube here:

This project (as written for Arduino UNO) requires one @ADAFRUIT NeoPixel Ring attached to Pin 5, one LSM303 Accelerometer/Magnetometer breakout board...

...and the following libraries:

Adafruit_NeoPixel Adafruit_Sensor.h Adafruit_LSM303.h Wire.h
